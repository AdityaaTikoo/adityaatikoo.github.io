WPApwn pronounced as WPA pwn, is an automated wifi cracker that captures WPA2 4-way handshake and cracks them via a dictionary attack.

It is still in development and I plan on adding the following features ->

  1. Denial of Service
  2. Offline brute forcing
  3. PMKID Attack Support
  4. Automated message when attack is completed. The tool is quite simple to use, just follow the instructions on my github page.
